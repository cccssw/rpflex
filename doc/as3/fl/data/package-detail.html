fl.dataClasses
 

The fl.data package contains classes that deal with data that is associated with a component.


Classes
 ClassDescription
 DataProvider The DataProvider class provides methods and properties that allow you to query and modify the data in any list-based component--for example, in a List, DataGrid, TileList, or ComboBox component.
 SimpleCollectionItem The SimpleCollectionItem class defines a single item in an inspectable property that represents a data provider.
 TileListCollectionItem The TileListCollectionItem class defines a single item in an inspectable property that represents a data provider.
